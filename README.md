# Основы Javascript
## Урок 10. Семинар. Объекты в JavaScript

Задание 1:

Дан объект numbers. Необходимо вывести в консоль все значения,
которые больше или равны 3.

```
const numbers = {
  keyin1: 1,
  keyin2: 2,
  keyin3: 3,
  keyin4: 4,
  keyin5: 5,
  keyin6: 6,
  keyin7: 7,
};
```
